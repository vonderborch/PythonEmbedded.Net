# Services

Service interfaces and implementations for process execution and other operations.

## Interfaces

- [IProcessExecutor](./Services/IProcessExecutor.md) - Interface for executing external processes

## Overview

Services provide:

- Standardized process result handling
- Logging and error handling for process operations
