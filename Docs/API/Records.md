# Records

Record types for data transfer and results.

## Records

- [PythonExecutionResult](./Records/PythonExecutionResult.md) - Result of Python code execution
- [ProcessExecutionResult](./Records/ProcessExecutionResult.md) - Result of process execution

## Overview

Records provide immutable data structures for:
- Execution results (exit codes, output, errors)
- Process execution results
- Package information (see [Models](./Models.md))
